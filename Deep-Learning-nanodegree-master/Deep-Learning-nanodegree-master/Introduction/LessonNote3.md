# Anaconda
## installing Anaconda
- install 
 ```
 conda install package_name
 ```
- install multiple packages
 ```python
 # example
 conda install numpy scipy pandas
 ```
 - list
```python
 conda list
 ```
 - search package
 ```python
 # example
 conda search beautifulsoup
 ```
## Managing environments
 ```
conda create -n env_name list of packages
 ```
